# Currículo Online — Filipe Silva Santos

Currículo pessoal desenvolvido em HTML5 e CSS3, hospedado via **GitHub Pages**.

🔗 **[Visualizar currículo online](https://filipesilvasantos.github.io/curriculo)**

---

## Sobre o projeto

Página estática que serve como currículo e portfólio pessoal. O layout é responsivo,
construído com o framework W3.CSS e ícones Font Awesome.

### Seções
- **Sobre Mim** — resumo profissional
- **Experiência** — histórico de atuação profissional
- **Educação** — formações acadêmicas e certificações
- **Projetos** — repositórios e trabalhos práticos
- **Habilidades** — stack técnica e idiomas
